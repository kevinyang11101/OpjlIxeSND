# 超市库存管理系统 java62

## 项目概述

本项目是基于 SpringBoot 3 和 Vue 3 的超市库存管理系统，旨在为超市提供一套全面、可靠的库存管理解决方案。

## 技术栈

### 后端技术

- **SpringBoot 3**：作为主要框架，用于构建后端服务。
- **Spring Security**：负责用户认证和授权。
- **MyBatis-Plus**：增强的数据库操作框架，简化开发流程。
- **MySQL**：稳定可靠的数据库系统，用于存储数据。
- **JWT**：实现用户身份的验证。
- **Knife4j**：API 文档工具，方便接口管理。
- **Spring AOP**：面向切面编程，提高代码的模块化。

### 前端技术

- **Vue 3**：前端框架，用于构建用户界面。
- **Vue Router**：负责前端页面的路由管理。
- **Pinia**：状态管理，便于维护全局状态。
- 组件化开发：采用结构化的前端设计，提高开发效率。

## 功能模块

### 系统功能

- 用户权限管理
  - 用户登录/注册
  - 角色权限管理
  - 菜单权限控制
- 商品管理
  - 商品分类管理
  - 商品信息管理
  - 商品批次质量管理
- 库存管理
  - 库存查询与管理
  - 入库管理
  - 出库管理
  - 库存预警
- 采购管理
  - 采购订单管理
  - 采购订单详情管理
  - 供应商管理
    - 供应商信息管理
    - 供应商评估系统
- 订单物流管理
- 数据分析与报表
- 系统管理
  - 系统日志记录

## 系统角色

系统主要角色包括管理员、普通用户等，各角色拥有不同的权限，协同完成超市库存管理工作。

## 运行环境

- 后端：Java 17+，SpringBoot 3
- 数据库：MySQL 5.7+
- 前端：Vue 3，现代浏览器
